# Lost Data Retrieval Project

This project is a simple demonstration of how deleted data is not immediately removed from a system and can still be traced or recovered.

## Objective
To understand how data persists on disk even after deletion and how basic scanning techniques can be used to identify traces of that data.

## Approach
- Created sample files with identifiable content  
- Deleted the files from the system  
- Used a Python script to scan and search for specific keywords in data  
- Explored recovery concepts similar to tools used in digital forensics  

## Tools Used
- Python

## Key Learning
Deleting a file does not permanently erase it from the system. The data remains until it is overwritten, which makes recovery possible and also introduces potential security risks.

## Note
This project is a simplified demonstration and focuses on understanding the concept rather than performing full-scale data recovery.
